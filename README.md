# Quote-Keeper

A simple and elegant web app to read, save, and search inspirational quotes.

## Features

- ✨ Display a random quote
- ➕ Add your own custom quotes with author name
- 🔍 Search through saved quotes
- 🧹 Clear input fields easily
- 📱 Fully responsive design
- 🔥 Powered by Firebase Firestore (Real-time database)

## Technologies Used

- **HTML5** & **CSS3**
- **Vanilla JavaScript**
- **Firebase Firestore** (v9 Compatibility)
- Responsive Design

## Project Structure
Quote-Keeper/
├── index.html
├── styles.css
├── script.js
└── README.md
text## How It Works

1. **Random Quote** - View a randomly selected quote from your collection
2. **Add Quote** - Type a new quote and author, then click "Add"
3. **Search** - Search through all saved quotes in real-time
4. **Clear** - Quickly reset the input fields

## Setup Instructions

1. Clone or download the project
2. Set up a Firebase project and enable Firestore
3. Replace the Firebase configuration in `script.js` with your own
4. Open `index.html` in your browser
